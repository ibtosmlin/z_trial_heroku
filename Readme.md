# Web アプリの練習
https://trial-heroku-ibtosm.herokuapp.com/

## 目的
- git連携
- flaskによるweb作成
- herokuによる公開

## 手順
1. アプリ作成
1. gitリポジトリに登録
1. github登録

1. herokuのアカウント作成
1. heroku cli インストール
1. heroku のアップデート
```
heroku -V
heroku update
```
1. heroku 空アプリ作成
1. DeployタブでGithub連携
